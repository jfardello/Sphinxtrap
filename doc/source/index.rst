<SphinxTrap>   
=============

.. container:: hero-unit hero-relocate-h1 hero-first-h2

    :icon:`bullhorn` A bootstrapped, responsive :btn:`Sphinx <http://sphinx-doc.org/>,icon-globe,` theme :icon:`picture`.

    Sphinxtrap is a minimalist bootstrap2 + Font Awesome sphinx theme that makes buttons and images out of rst roles.


Features
--------

.. container:: pull-left clearfix 

    * :icon:`ok` Remote Google Fonts 
    * :icon:`ok` Google Analytics tracking
    * :icon:`ok` Font Awesome icon role for Sphinx
    * :icon:`ok` Bootstrap button role for Sphinx
    * :icon:`ok` Easily customizable via css
